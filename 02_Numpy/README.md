# Data Analysis Using NumPy

This folder contains materials and notebooks for learning data analysis using NumPy. The content is written for beginners and intermediate learners and includes conceptual explanations, hands-on examples, and practical exercises.

**What you'll learn**
- Why NumPy is important for data analysis and scientific computing.
- How NumPy arrays differ from Python lists.
- Creating, indexing, slicing, and reshaping arrays.
- Vectorized operations, broadcasting, and performance tips.
- Common numerical routines: aggregations, statistics, and linear algebra basics.

**Folder structure**
- `Programs/` — Contains the notebooks and runnable exercises.
- `NumPy_Fundamentals-checkpoint.ipynb` — Hands-on examples demonstrating core NumPy features.
- `Introductions-checkpoint.ipynb` — Short introduction notes and quick-start guidance.

**How to open and run**
1. Install requirements (if needed). NumPy is required; install with:

```bash
pip install numpy
```

2. Open the notebooks in Jupyter or VS Code.
3. Run cells in order. Use a Python 3.8+ kernel for best compatibility.

**Suggested learning path**
1. Read the `Introductions` notebook to get background and objectives.
2. Work through `NumPy_Fundamentals` and run each cell, experimenting with different inputs.
3. Try the exercises and modify arrays to reinforce learning.

**Exercises & Projects**
- Compute summary statistics (mean, median, std) for sample arrays.
- Implement matrix multiplication and compare with `np.dot`.
- Create a small data-cleaning pipeline using boolean masking and `np.where`.

**Resources**
- Official NumPy docs: https://numpy.org/doc/
- Tutorials: https://numpy.org/learn/
