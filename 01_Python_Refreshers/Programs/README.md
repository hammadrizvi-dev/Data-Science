# Python Programs

This folder contains the Python refresher exercises organized for easier browsing.

## Available Programs

- [conditional_statements.py](conditional_statements.py) - Practice with conditional logic.
- [data_types.py](data_types.py) - Examples of Python data types.
- [dictionaries.py](dictionaries.py) - Dictionary operations and usage.
- [error_handling.py](error_handling.py) - Try/except and exception handling.
- [file_handling.py](file_handling.py) - Reading and writing files.
- [functions.py](functions.py) - Function definitions and reuse.
- [input_and_type_conversion.py](input_and_type_conversion.py) - User input and type conversion.
- [iterators_and_generators.py](iterators_and_generators.py) - Iteration patterns and generators.
- [lambda_and_comprehension.py](lambda_and_comprehension.py) - Lambda expressions and comprehensions.
- [lists.py](lists.py) - List operations and examples.
- [loops.py](loops.py) - For and while loops.
- [modules_and_imports.py](modules_and_imports.py) - Importing modules and packages.
- [oops.py](oops.py) - Object-oriented programming examples.
- [operators.py](operators.py) - Arithmetic and comparison operators.
- [strings.py](strings.py) - String manipulation examples.
- [tuples_sets.py](tuples_sets.py) - Tuples and sets.
- [variables.py](variables.py) - Variable declaration and usage.

## Suggested Order

1. Start with [variables.py](variables.py)
2. Move to [data_types.py](data_types.py)
3. Practice with [operators.py](operators.py)
4. Explore [lists.py](lists.py) and [dictionaries.py](dictionaries.py)
5. Continue with [functions.py](functions.py) and [oops.py](oops.py)
